# sAPI for bluepill

Port from sAPI library to STM32F103C8 bluepill board.

sAPI is a simple API that acts as a HAL (Hardware Abstraction Layer) for microcontrollers used in the [project CIAA](http://www.proyecto-ciaa.com.ar/).
